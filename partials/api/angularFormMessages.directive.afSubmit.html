afSubmit
directive in module angularFormMessages

Description

This directive handles the submit event.

  1. All existing validations are cleared.
  2. The form is checked for client side validations.
  3. If valid, the submit callback is being called. When this returns a rejecting promise, validation messages are broadcasted.

While submitting, $scope.isSubmitting is set to true.

Usage

as attribute
<ANY af-submit>
   ...
</ANY>
