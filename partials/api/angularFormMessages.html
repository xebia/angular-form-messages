The Angular Form Messages module. Contains all basic functionality to make Angular Form Messages possible.

Directives

  • afField - Registering a form element.
  • afMessageLabel - Showing a translated message.
  • afMessages - Showing messages.
  • afSubmit - Handles the form submit.
  • afSubmitButton - Disables the form submit button and sets a flag on the scope while submitting.

Services

  • afMessageService - Contains all shared utility functions that are used by the directives. The provider does app wide configuration.
